# Suiseki

The musical trails of Suisei.

**<https://suiseki.suisei.moe>**

## Features

- See the evolvement of Suisei's voice by time, by listening to the songs
- Search for a certain song (try this: <https://suiseki.suisei.moe/?q=comet>)
